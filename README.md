# RENTER asunnonvälitys

## Sovelluksen toiminnot

* Sovelluksessa käyttäjät pystyvät jakamaan ilmoituksia vuokrattavista asunnoista. Ilmoituksissa lukee tarvittavat tiedot asunnon koosta, sijainnista ja hinnasta.
* Käyttäjä pystyy luomaan tunnukset ja kirjautumaan sisään sovellukseen.
* Käyttäjä pystyy katsomaan, sekä omia että muiden ilmoituksia.
* Käyttäjä pystyy lisäämään, muokkaamaan ja poistamaan omia ilmoituksia.
* Käyttäjä pystyy lisäämään kuvia ja tekstiä ilmoituksiinsa.
* Käyttäjä pystyy antamaan arvosanan (1-5) ja tekstimuotoisen palautteen toiselle käyttäjälle, mikäli he ovat asioineet keskenään.
* Sovelluksessa on käyttäjäsivut, jotka näyttävät oman ilmoitushistorian, sekä oman (saatuihin arvosanoihin) perustuvan luokituksen, sekä muilta käyttäjiltä tulleen palautteen, että omat annetut palautteet.
* Käyttäjä pystyy hakemaan ilmoituksia hakusanoilla ja neliöiden, huonemäärän, sijainnin, hintaluokan, sekä ilmoittajan luokituksen perusteella.
* Käyttäjä pystyy valitsemaan ilmoituskohteelleen, yhden tai useamman luokituksen. 

Pääasiallinen tietokohde on asuntoilmoitukset ja toissijainen tietokohde on käyttäjien arvosanaluokitukset ja palautteet.
